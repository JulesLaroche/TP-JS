README - Travaux Pratiques en JavaScript : Manipulation du DOM
Ce projet contient une série de Travaux Pratiques (TP) en JavaScript conçus pour vous apprendre à manipuler le DOM (Document Object Model) et à créer des interactions dynamiques dans les pages web. Chaque TP se concentre sur des concepts spécifiques de la manipulation du DOM.

Clonage du Projet : Clonez ce dépôt sur votre ordinateur en utilisant la commande git clone

Sélection du TP : Chaque TP est dans un dossier séparé. Choisissez le TP sur lequel vous souhaitez travailler.

Ouverture dans le Navigateur : Ouvrez le fichier index.html du TP sélectionné dans un navigateur web pour voir l'exemple de base.

Modification du Code : Modifiez le fichier script.js pour accomplir les tâches décrites dans le TP.

Test et Expérimentation : Utilisez le navigateur pour tester votre code et expérimenter avec la manipulation du DOM.
